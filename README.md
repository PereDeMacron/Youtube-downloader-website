youtube downloader in js-html-css

how it work:
1- in vsc terminal type "node index.js"
2- open the index.html
3- get a yt link example ("https://www.youtube.com/watch?v=fbodK7Vv-Sw")
4- write the yt link in the URL box
5- click on convert
6- watch your video
